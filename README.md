
# Besa Ventures E-commerce Demo (No Payment)
- Real content inserted
- Git-ready
- File upload support (Gerber)
- Email SMTP integration

Run backend:
```
cd backend
npm install
export ADMIN_TOKEN="token"
npm start
```

Push to GitHub:
```
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin YOUR_REPO_URL
git push -u origin main
```
